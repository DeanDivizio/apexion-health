## `legRaise` evidence

Defaults location: `lib/gym/exercises.ts`

### Base targets (abs + hip flexors)

- **Code pointer**: `legRaise.baseTargets`
- **Rationale**: Leg raises combine hip flexion with abdominal stabilization/anti-extension; hip flexors contribute meaningfully alongside rectus abdominis.
- **Sources**
  - Rectus abdominis vs rectus femoris activity during leg raises (ratio changes with angle): `https://ncbi.nlm.nih.gov/pmc/articles/PMC4395661/`
  - Abdominal + hip flexor activation across exercises: `https://pubmed.ncbi.nlm.nih.gov/9118976/`
- **Confidence**: Medium.
